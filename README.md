# Charles-Proxy 


При тестировании веб-сервисов иногда может понадобиться инструмент для перехвата и изменения трафика, тестирования низкой скорости интернета и т.д. Ниже приведено несколько вариантов использования Charles Proxy.
* [Изменение количества добавляемого в корзину товара через брейкпоинты](https://drive.google.com/file/d/1zIxOvMFaytvmhvyiQNpEouXmZVAvvVNs/view?usp=sharing)
* [Изменение отображаемого количества товаров в корзине через рерайт](https://drive.google.com/file/d/1qn-zPVDN7Zh8jfYw7alzUM4bolUFk3R_/view?usp=sharing)
* [Изменение статус-кода ответа сервера](https://drive.google.com/file/d/1M4vJb69Qm4UNLNyOG89jvn7YqwmQyl2E/view?usp=sharing)
* [Перенаправления запроса на другой URL](https://drive.google.com/file/d/1DnNdYj0yLClWmVI1DBZ3UAzM1f1DKcuS/view?usp=sharing)

# Charles-Proxy 


При тестировании веб-сервисов иногда может понадобиться инструмент для перехвата и изменения трафика, тестирования низкой скорости интернета и т.д. Ниже приведено несколько вариантов использования Charles Proxy.
* [Изменение количества добавляемого в корзину товара через брейкпоинты](https://drive.google.com/file/d/1DLLpNDopcF3T6-apOhDu1AUdi5eFa6nW/view?usp=sharing)
* [Изменение отображаемого количества товаров в корзине через рерайт](https://drive.google.com/file/d/1x77eLtj2G6JuwG4mqaBK1OdK_9iuIM8y/view?usp=sharing)
* [Изменение статус-кода ответа сервера](https://drive.google.com/file/d/1OxAuLducOn3zg4zRuJHGY88CfqItCIPj/view?usp=sharing)
* [Перенаправления запроса на другой URL](https://drive.google.com/file/d/1_tOApdGGeeiYTOwkGTrKt79M7HmEPzDZ/view?usp=sharing)
